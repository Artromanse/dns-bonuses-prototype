# PROJECT — Бонусы (прототип)

## Живая ссылка
**https://bonuses-seven.vercel.app**

## Деплой
- Путь: сервис (Vercel)
- Источник: GitHub-репозиторий `Artromanse/dns-bonuses-prototype`, ветка `main`
- Проект в Vercel: `romanse07-5904s-projects/bonuses`
- Автопересборка: включена — каждый пуш в `main` пересобирает продакшн (~1–2 мин)
- SPA-маршрутизация: не требуется — прототип это один статический `index.html` без клиентского роутинга

## Как обновить
```
git add -A && git commit -m "..." && git push origin main
```
Vercel подхватит пуш автоматически. Локальная разработка — открыть `index.html` в браузере.
