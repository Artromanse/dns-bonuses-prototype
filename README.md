# Бонусы — прототип (Кабинет покупателя DNS)

🔗 **Живая версия: https://artromanse.github.io/dns-bonuses-prototype/**

HTML-прототип мобильного экрана «Бонусы» из кабинета покупателя DNS
с центром уведомлений (полноэкранная мобильная модалка по иконке чата).
Собран из Figma-макетов (node `8342:63174`, `1808:16767`).

## Просмотр

Открыть ссылку выше, либо локально — `index.html` в браузере.

## Состав

- `index.html` — экран целиком (на CSS-токенах DS, шрифт PT Sans)
- `assets/` — векторные иконки и логотипы (SVG, выгружены из Figma)
- `assets/notif-groups/` — иконки групп уведомлений (см. «Компоненты» ниже)

## Состояния

Кнопки реализованы со всеми состояниями DS (default / hover / pressed),
значения — из токенов дизайн-системы DNS.

## Компоненты

- **Центр уведомлений — пересобран (2026-07-02)** под актуальный макет Figma:
  [Выбор города / Смена города / крестик в попап. Макет](https://www.figma.com/design/c90ULUDPYwSjpydLlmitt5/?node-id=6959-1526)
  (файл `c90ULUDPYwSjpydLlmitt5`, node `6959:1595`).
  Старая версия с табами («Сообщения» / «Уведомления») и карточками-уведомлениями
  заменена на **двухуровневую навигацию**:
  - **Уровень 1** — список групп уведомлений (Заказы, Системные сообщения, Мои подписки,
    Корзина и избранное, Рекомендации, Скидки и акции, Чат-бот (AI), Клуб DNS):
    иконка (44×44, фон `Background/disabled #eae8e8`) + заголовок/время + превью-текст
    (обрезается многоточием) + опционально счётчик непрочитанных (белая пилюля,
    текст `Text/brand #fc8507`) или Wobbler mini «Новое» (`Background/success-main #00be70`).
  - **Уровень 2** — клик по группе проваливается в её детальный список (по датам,
    карточка = иконка + текст + кнопка «Перейти к заказу»); реальным контентом
    заполнена только группа «Заказы» (единственная с готовым макетом), для остальных —
    пустое состояние-заглушка. Хедер при этом переключается: крестик → стрелка «Назад»,
    заголовок «Центр уведомлений» → название группы.
  - Хедер (`Modal header`): слева — закрыть/назад (переключается по уровню), справа —
    **«Прочитать всё»** (не «выбрать всё» — уточнение 2026-07-02): по клику у всех групп
    с бейджем (счётчик или Wobbler «Новое») бейдж исчезает.
