# Бонусы — прототип (Кабинет покупателя DNS)

🔗 **Живая версия: https://artromanse.github.io/dns-bonuses-prototype/**

HTML-прототип мобильного экрана «Бонусы» из кабинета покупателя DNS
с центром уведомлений (полноэкранная мобильная модалка по иконке чата).
Собран из Figma-макетов (node `8342:63174`, `1808:16767`).

## Просмотр

Открыть ссылку выше, либо локально — `index.html` в браузере.

## Состав

- `index.html` — экран целиком (на CSS-токенах DS, шрифт PT Sans)
- `assets/` — векторные иконки и логотипы (SVG, выгружены из Figma)

## Состояния

Кнопки реализованы со всеми состояниями DS (default / hover / pressed),
значения — из токенов дизайн-системы DNS.
